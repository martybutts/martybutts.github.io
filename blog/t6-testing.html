Explain to a non-tech friend what 'scope' is and how it works in JavaScript.

In JavaScript, objects and functions are also variables. The concept of scope is all to do with how and where you use variables. If a variable is used outside of a function it has 'global' scope, and can be accessed by other code throughout the program. If a variable is used inside a function it has 'local' scope, and can only be accessed within that function.
