What is meant by boxifying design?

Boxiying design is an approach to web page design where all of the elements are first considered as being compartmentalised into boxes. This adds a sense of logic to the organisation of the page, and also allows you to redesign the page more easily by moving around the boxes.

What is the box model?

This is ther term we use when talking about the design and layout of web pages. If we think about each html element having a box wrapped around it, this can help when considering editing the page. Each box contains margins, borders and padding around the actual content, all of which can be adjusted.
